Aplicaciones de serie

Ubuntu trae de serie una cantidad importante de aplicaciones, que va actualizando y cambiando con cada nueva versión. Algunas de las apps que vienen de serie son:

  • Firefox como navegador web.
  • Empathy como cliente de mensajería instantanea.
  • Gwibber como cliente de redes sociales.
  • Thunderbird como cliente de correo.
  • Totem como reproductor multimedia.
  • Banshee como reproductor de música.
  • Shotwell como gestor y editor de fotografía.
  • Transmission como cliente de torrents.
  • Brasero como grabador de discos.
  • Libre Office como suite ofimática.
  • Gedit como editor de texto.
  • Terminal como emulador de terminal-

Pero estas no son las únicas.

No obstante podemos desinstalar e instalar nuevas apps a través de sus paquetes, agregando nuevos repositorios o a través del Centro de Software de Ubuntu.

Agregar y actualizar aplicaciones

Para instalar una nueva aplicación existen varios métodos. O bien descargar un paquete de la aplicación y ejecutarlo, o la otra opción es agragar un repositorio. Los repositorios tienen la ventaja de que más tarde esa aplicación se actualizará junto con el resto de aplicaciones. Por ejemplo el repositorio de la versión beta de Firefox es:

ppa:mozillateam/firefox-next

Este código tenemos dos opciones de agregarlo:

  • Terminal: Copiar en terminal:
    sudo add-apt-repository ppa:mozillateam/firefox-next
  • Fuentes de software: Agregarlo al menú de preferencias del sistema: fuentes de software.

En los repositorios hay varias aplicaciones, para instalar una de las aplicaciones de un repositorio que tengamos podemos hacerlo desde la terminal escribiendo: sudo apt-get install nombre_aplicacion o bien sudo aptitude install nombre_aplicacion.

Centro de Software

El Centro de Software Ubuntu fue la respuesta de Canonical a las quejas por la dificultad de instalar software. Este centro permite instalar, desinstalar y buscar nuevas aplicaciones de una forma intuitiva y sencilla ya que es todo por interfaz gráfica y cuenta con imágenes, buscador, zona de comentarios...

WINE

Para aquellos usuarios que vienen de Windows y no encuentran un software alternativo en Linux siempre pueden recurrir al uso de WINE que permite ejecutar bastantes programas pensados para Windows en Ubuntu y otras distribuciones de Linux.
