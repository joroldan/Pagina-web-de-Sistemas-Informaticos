Introducción

Ubuntu es una distribución de GNU-Linux, un sistema operativo gratuito y de código abierto. Está disponible sobre las arquitecturas x86, AMD64 (x86-64) y ARM. Antiguamente también soportaba PowerPC y extraoficialmente SPARC, IA-64 y PlayStation 3 (CELL).

Núcleo

Comparte núcleo con el resto de distribuciones de GNU-Linux. Así su núcleo Linux, basado en UNIX es de tipo monolítico. La última distribución de Ubuntu usa la versión del núcleo 3.1

Idiomas

Ubuntu está disponible en más de 130 idiomas, lo que hace que la gran mayoría de usuarios puedan usarlo en su idioma nativo. Además si no estuviera disponible en tu idioma puedes colaborar con la traducción.

Distribuciones derivadas

Al igual que Ubuntu deriva de Debian, existen otras distribuciones que derivan de Ubuntu. Las más significativas son:

  • Lubuntu, utiliza como entorno de escritorio LXDE y consume bastantes menos recursos.
  • Kubuntu, utiliza como entorno de escritorio KDE.
  • Edubuntu, diseñado para entornos escolares (a partir de la versión 8.04 es un paquete agregado dejando de ser una distribución aparte).
  • Xubuntu, utiliza como entorno de escritorio Xfce.
  • Ubuntu Netbook Edition, para netbooks (descontinuada).
  • Kubuntu Netbook Edition, para netbooks con interfaz KDE (descontinuada).
  • Ubuntu Studio, diseñada para el trabajo con multimedia. No reconocida oficialmente por Ubuntu.
  • Tiflobuntu, una versión de Ubuntu para personas ciegas y con visión reducida. Funciona mediante línea braille.
  • Ubuntu Studio, diseñada para el trabajo con multimedia. No reconocida oficialmente por Ubuntu.

Edubuntu, Kubuntu, Lubuntu y Xubuntu son proyectos oficiales.

Requisitos

Ubuntu permite probar la distribución sin necesidad de realizar ninguna instalación o modificación en el sistema por medio de un live-CD o live-USB. Además, desde las últimas versiones permite probar Ubuntu a través del navegador. La 11.04 era accesible completamente online y la versión 11.10 tiene un tour online muy limitado para hacerse una breve idea de cómo es.

Los requisitos mínimos «recomendados», teniendo en cuenta los efectos de escritorio, deberían permitir ejecutar una instalación de Ubuntu.

Obtención

  • Descargar la imagen ISO por descarga directa o torrent y montarla en un CD/DVD/USB y proceder a la instalación o a su uso como live-image.
  • Descargar una imagen de 10Mb que permite una posterior instalación a través de internet.
  • Descargar el instalador de Windows, que nos permite configurar una nueva partición con Ubuntu de una forma sencilla.
  • Si disponemos de una versión anterior de Ubuntu, podemos proceder desde el gestor de actualizaciones del propio sistema.
  • Adquirir un equipo con Ubuntu instalado de serie.
  • Antiguamente Canonical enviaba CDs con Ubuntu, pero este servicio acabó en 2010.
