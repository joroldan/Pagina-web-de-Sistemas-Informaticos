Introducción

Ubuntu es una distribución de GNU-Linux desarrollada por Canonical y una comunidad de usuarios. Utiñiza el núcleo Linux y está basado en Debian. Su eslogan es Linux for human beings (‘Linux para seres humanos’).

Esta distribución está orientada al usuario medio, con un fuerte enfoque a la facilidad y a la experiencia de usuario. Se estima que prácticamente la mitad de las versiones de Linux instaladas corresponden a esta distribución.

Origen

Su nombre proviene de la ideología sudafricana Ubuntu ("Igualdad y lealtad hacia otros").

El proyecto base es una bifurcación de Debian con el objetivo de crear un sistema más actualizado, una nueva versión cada seis meses.

Ubuntu comenzó su andadura el 20 de octubre de 2004 con su primera versión: Ubuntu 4.10

Desarrollo

Ubuntu usa principalmente software libre, aunque con excepciones en determinados controladores y firmare del núcleo, además de software no libre en sus repositorios.

Al estar basado en debían, ambos usan los mismos paquetes de software ".deb" y las mismas herramientas de paquetes APT ".dpkg", aunque no siembre hay compatibilidad binaria entre ambas distribuciones. Antes de cada lanzamiento se importan los paquetes desde Debian y se aplican modificaciones específicas de Ubuntu.

Fases del desarrollo

Cada versión de Ubuntu pasa por un ciclo de desarrollo. El ciclo tiene las siguientes fases:

  • Alpha:Son dos versiones aún muy inestable y no incluyen todas las novedades.
  • Beta:Son dos versiones, algo más estables con todas las novedades que se esperan incluir.
  • RC:Una semana antes sale esta versión, que se supone estable y completa, pero que aún se prueba en busca de algún error que no se haya visto.
  • Final:Esta es la versión que se hace disponible de forma masiva y se supone estable y completa.

Se puede consultar el ciclo de desarrollo estimado de las futuras versiones en la siguiente web (enlace externo)

Lanzamientos

Ubuntu sigue un estricto ciclo de versiones, lanzando una nueva versión estable cada seis meses.

Numeración

La numeración de la versiones es muy simple, las dos primeras cifras corresponden al año de lanzamiento, luego se inserta un punto y las dos siguientes cifras corresponden al mes de lanzamiento.

Así la versión 11.10 fue lanzada en el año 11 (2011) y en el mes 10 (octubre).

Nombre

El code-name de cada versión está compuesto por un adjetivo y un nombre de animal, empezando ambos por la misma letra. Desde la cuarta versión (6.06) además se sigue un orden alfabético.

Soporte

Cada versión tiene un soporte técnico con actualizaciones de seguridad durante 18 meses.

No obstante cada dos años Canonical lanza una versión LTS (Long Term Support) que ofrece 3 años de soporte para escritorio y 5 para servidor, aunque recientemente se ha anunciado que la próxima versión LTS tendrá un soporte de 5 años en todos los campos.
